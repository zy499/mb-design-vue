## zh-CN

指定 `mode` 为 `button` 来使用带按钮的数字输入框。

---

## en-US

Specify `mode` as `button` to use a numeric input box with buttons.

---

```vue
<template>
  <a-input-number :style="{width:'320px'}" placeholder="Please Enter" :default-value="500" mode="button" class="input-demo" />
</template>
```
